# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/portfolio.git
```

## 2️⃣ Move into Project Folder

```bash
cd portfolio
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm run dev
```
